# CurrencyExchange
Сервис расчета курса валюты
При локальном развертывании приложения в IIS - базовая страница Swagger
2 метода.
1) "api/coordinates" - Получение квадратна окружности по 2 координатам
2) "api/exchangerate" - Получение курса валюты по 2 координатам(квадратну) - соответствующему определенной дате
